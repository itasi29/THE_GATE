【操作説明】
※Xボックスコントローラー基準です。

Aボタン：決定、ジャンプ
Bボタン：キャンセル、掴む、離す
Xボタン：ダッシュ(長押し)
Rボタン：視点180度回転(後ろを向く)

Rトリガー：ゲート生成(長押し可能)
Lトリガー：ゲート生成(長押し可能)

右スティック：移動
左スティック：カメラ
